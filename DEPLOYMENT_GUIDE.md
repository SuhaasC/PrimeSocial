## ЁЯФз Hostinger Upload Steps:

### **Step 1: Access File Manager**
1. **Login** to Hostinger control panel
2. **Go to** "File Manager" or "Files"
3. **Navigate** to your domain's root directory (usually `public_html`)

### **Step 2: Clear Existing Files**
1. **Delete** all existing files in `public_html`
2. **Keep** any important files you want to preserve
3. **Note**: This will remove your current WordPress site

### **Step 3: Upload New Files**
1. **Upload** the entire contents of your `dist` folder
2. **Ensure** `.htaccess` is uploaded (this is crucial!)
3. **Verify** all files are in the root directory

### **Step 4: Set Permissions**
1. **Set** `.htaccess` to 644 permissions
2. **Set** folders to 755 permissions
3. **Set** files to 644 permissions
